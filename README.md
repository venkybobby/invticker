# Forge - Personal Institutional Research Desk

Next.js 14 + TypeScript dashboard for institutional-grade investment analysis.

Built around 15 specialized models (Goldman Sachs, KKR, Bridgewater, JPMorgan, Citi, etc.) with live data tool calling planned.

## Current Status (Week 1 Complete)
- Clean sidebar navigation
- Fully interactive Profile section
- Dynamic Portfolio holdings table
- Financial Modeling Inputs (DCF/LBO/M&A ready)
- Output Center with prompt generation

## Getting Started

```bash
npm install
npm run dev
```
